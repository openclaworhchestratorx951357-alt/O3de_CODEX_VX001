# frontend

React and Vite frontend for the O3DE Agent Control app.

## Requirements

- Node.js 20+
- npm

## Install

1. Change into the frontend directory.
2. Run npm install.

## Run

1. Change into the frontend directory.
2. Run npm run dev.

## Build

1. Change into the frontend directory.
2. Run npm run build.

## Current UI

- agent control panels
- approval queue
- layout header
- task timeline
